# Afaq Coverage Test

هذا اختبار أولي لفحص هل موقع العميل داخل زون أو خارجها.

## الملفات
- index.html: صفحة الخريطة والفحص.
- zones.geojson: الزونات المحولة من ملف KML.
- points.geojson: نقاط FAT/الكابينات إن وجدت.

## التشغيل
بسبب أن المتصفح يمنع fetch من الملفات المحلية أحيانًا، شغل سيرفر بسيط داخل هذا الفولدر:

```bash
python -m http.server 8000
```

ثم افتح:

http://localhost:8000

## الفكرة
OpenStreetMap = الخريطة الخلفية.
Leaflet = عرض الخريطة والزونات.
GeoJSON = ملف حدود الزونات.
Turf.js = فحص هل نقطة العميل داخل الزون.
